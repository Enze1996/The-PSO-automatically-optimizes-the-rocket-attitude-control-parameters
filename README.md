1.代码作用说明：
    通过粒子群优化 (PSO) 算法优化 PID 控制器参数，可以显著提高火箭姿态控制系统的稳定性和精确性。该 C++ 架构实现了 PSO 算法的粒子群初始化、粒子更新、个人和全局最佳更新等基本操作，并通过目标函数计算姿态控制误差，最终优化 PID 参数。在火箭姿控应用中，PSO 是一个有效的自动化优化工具。
